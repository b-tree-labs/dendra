# USPTO FORM SB/16 — PROVISIONAL APPLICATION COVER SHEET

fields, export to PDF, attach as first sheet of the filing packet.

[uspto.gov/sites/default/files/documents/sb0016.pdf](https://www.uspto.gov/sites/default/files/documents/sb0016.pdf).

information via a structured form — the PDF is included mostly
as a paper-trail record.
