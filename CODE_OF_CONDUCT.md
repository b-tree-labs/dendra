# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In summary:

- **Be kind.** Treat others with dignity. Disagreement is fine;
  personal attacks are not.
- **Be specific.** Feedback on code, behavior, or decisions is far
  more useful when concrete.
- **Be inclusive.** The project welcomes contributors regardless of
  background, experience level, or identity.

Unacceptable behavior includes harassment, discriminatory language,
sustained disruption, and doxxing. Reports can be sent to
`ben@b-treeventures.com`. The maintainer will review and respond
within 5 business days.

Consequences for violations range from a warning to permanent ban
from the project, at the maintainer's discretion, scaled to
severity.

See the [full Contributor Covenant v2.1 text](https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md)
for the complete terms this project adopts.
